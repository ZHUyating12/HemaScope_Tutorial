HemaScope Tutorial

1 Installing HemaScopeR:

1.1 Quick Installation(On Conda)

1.1.1 1. Install R4.3.3, python3.11.4

HemaScopeR is a package written in R and Python.

  • Create a new conda environment,install R4.3.3 and python 3.11.4
conda create --name HemaScopeR_env
conda activate HemaScopeR_env
conda install R-base=4.3.3
conda install python=3.11.4
